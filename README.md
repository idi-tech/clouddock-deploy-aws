# CloudDock Deploy — Dockerized Web Application Deployment on AWS

CloudDock Deploy is a lightweight DevOps project designed to demonstrate how a containerized web application can be built, versioned, and prepared for deployment on AWS infrastructure.

## Project Goals

- Build a lightweight web application
- Containerize the application using Docker
- Manage source code with Git and GitHub
- Prepare the project for CI/CD automation
- Provision AWS infrastructure using Terraform
- Deploy the application to an EC2 instance

## Tech Stack

- Linux
- Git & GitHub
- Docker
- AWS EC2
- Terraform
- GitHub Actions
